# CHANGELOG

## Unreleased

## v0.4.6 - 2026-06-22

+ [新增] NewToken `gpt-image2-1k` / `gpt-image2-2k` / `gpt-image2-4k` 改为按官方异步图片任务协议走 `/v1/videos` 创建和轮询，`*_sync` 模型继续使用同步图片接口。
+ [优化] 画布视频提交与视频创作台校验对齐，统一空提示词拦截、参考素材校验、比例/清晰度/秒数/声音/水印参数归一化。
+ [优化] Cai / NewToken / duomiapi 视频参考图片、视频和音频在提交前会上传为公网 HTTPS URL，减少本地 `blob:` / dataURL 导致的上游超时。
+ [优化] 画布缩小时保持生成输入面板可读，并提高视频输入区最小高度。
+ [修复] 修复画布视频提交时误用未定义变量导致 `node is not defined` 的问题。
+ [修复] 视频最大生成时长统一限制为 15 秒，避免提交上游不支持的更长时长。

## v0.4.5 - 2026-06-20

+ [新增] 渠道调用格式新增 NewToken 异步接口，支持常用 NewToken 视频模型一键填入，并按模型族适配异步视频创建、轮询和参考素材字段。
+ [优化] 配置页去除 NewToken 底层协议说明文案，保持调用格式选择区简洁。

## v0.4.4 - 2026-06-20

+ [优化] 画布视频生成面板按 C-AI Apifox 文档动态展示模型能力，Seedance 支持首尾帧和全能参考，Grok 模型仅展示其支持的文生/图生/图片参考能力。
+ [优化] Cai 视频任务 payload 按模式拆分首尾帧、全能参考和普通图片参考，避免多图参考被误判为首尾帧。
+ [修复] 修复视频生成输入面板在参数较多或模型名较长时底栏遮挡输入框的问题，并补齐图片/视频参数弹层在配置节点中的全宽样式。

## v0.4.3 - 2026-06-20

+ [新增] 画布自动整理新增视口自动对焦（Auto-Focus）功能，一键将整理后的树状推演图缩放居中于屏幕。
+ [新增] 提示词面板集成魔法棒一键优化提示词（🪄）功能，可自动丰富高级电影级细节修饰词。
+ [优化] 提示词面板底部控制栏像素级重构：大圆角纯白微阴影卡片设计，平铺无边框输入区，并压缩为极简单行底栏。
+ [优化] 视频参数设置和图片参数设置重构为组合胶囊按钮（如视频的比例、时长、清晰度组合展示；图片的清晰度、比例组合展示），极大扩展可用宽度。
+ [优化] 画布自动整理拓扑间距调优，将列间距增大至 `280` 像素，行间距增大至 `100` 像素，避免复杂拓扑折线重叠。
+ [优化] 画布视频连线流动特效 (Flowing Glow) 以及输入框上方参考卡片 Hover 遮罩删除交互。

## v0.4.2

+ [新增] 视频生成接入 Cai 专用接口，按模型类型适配 `/v1/videos` 与 `/v1/video/generations` 异步创建和轮询路径。
+ [新增] 画布视频节点保存异步任务 ID，超时、刷新或网络中断后可手动拉取已完成的视频结果。
+ [新增] 增加 AI 请求代理，拉取模型、渠道诊断、生图、音频和视频请求可通过 Next.js 服务端转发，降低跨域和 403 问题。
+ [新增] 首页增加动态视觉效果和画布节点演示墙，强化 C-AI 画布生成、连接和连续推演的产品表达。
+ [优化] 视频创作台和画布视频任务轮询最长等待时间调整为 30 分钟，适配长耗时上游任务。
+ [优化] 渠道设置中原 OpenAI JSON 调用格式调整为 Cai 专用接口展示，并优化视频超时与公网素材 URL 提示。
+ [修复] 修复视频节点生成成功后无法拖动、播放时误触发节点拖拽的问题。
