# CHANGELOG

## Unreleased

## v0.4.18 - 2026-06-30

+ [修复] 页面和接口响应增加 no-store 缓存控制，并让旧版同步生图代理请求快速提示刷新，避免浏览器或 CDN 继续使用旧前端导致 524。

## v0.4.17 - 2026-06-30

+ [修复] AI 请求代理转发同步图片生成/编辑时改为本地代理任务轮询，避免上游已成功但本地长连接被边缘网关 524 截断。

## v0.4.16 - 2026-06-29

+ [新增] 渠道调用格式新增 Lingdong，内置灵动模型列表，并适配其图片生成和异步视频创建/查询协议。

## v0.4.15 - 2026-06-28

+ [优化] 异步视频任务轮询到上游公开视频 URL 后，画布和视频创作台直接显示结果，不再等待前端完整下载视频文件，减少上游已成功后的页面延迟。
+ [新增] 补充服务器本地构建升级脚本，便于部署环境按当前代码重新构建 Docker 镜像并刷新缓存。

## v0.4.14 - 2026-06-25

+ [修复] 生图/视频工作台生成中或上游失败的记录支持停止本地任务并删除，画布删除节点时也会中断关联生成请求，避免异步任务完成后把已删除内容写回。

## v0.4.13 - 2026-06-25

+ [优化] NewToken 图片、视频和音频参考素材改为通过服务端代理上传到 `aimh8` 媒体接口，默认内置上传 Key，并统一使用返回的 OSS 公网 URL 提交给上游。

## v0.4.12 - 2026-06-25

+ [优化] 视频创作台左侧生成记录补充生成时间显示，便于按时间回看历史任务。

## v0.4.11 - 2026-06-25

+ [修复] 云端个人渠道保存 Key 支持在服务端数据目录自动生成并持久化加密密钥，Docker 默认挂载账号数据目录，避免登录后保存渠道因缺少密钥或容器重启而失效。
+ [修复] 在线 Agent 工具对话在上游不兼容 Responses 工具 schema 时自动降级到 Chat Completions 工具格式，避免出现 `Bad input` 导致无法聊天。
+ [优化] 默认 Docker Compose 部署改为使用本地 Dockerfile 编译镜像，线上更新可直接从当前代码构建，不再依赖远端 latest 镜像缓存。

## v0.4.10 - 2026-06-25

+ [优化] 生图和视频提交前会将 `@图片1`、`@视频1`、`@音频1` 规范为编号文本，并在提示词前追加参考素材编号说明，确保上游按实际素材字段理解引用关系。
+ [优化] Cai、NewToken、Duomi 和 Seedance 视频参考素材提交前强制转换为公网 HTTPS URL，并通过 `HEAD` 或 `GET Range` 自检可读性，失败时前端直接拦截。
+ [优化] 调用格式、云端渠道、Agent 面板和鉴权错误提示统一去掉 API 字样，保留更简洁的 Cai、NewToken、Duomi 和 Key 表达。

## v0.4.9 - 2026-06-24

+ [优化] 视频创作台提示词框支持 `@` 引用已添加的参考图片、视频和音频素材，并继续随请求传给视频任务。
+ [优化] 画布、生图工作台和视频工作台的 `@` 引用按钮避免触发输入框失焦，候选菜单点击后稳定弹出。
+ [新增] 视频创作台生成记录支持首帧缩略图，加入“我的素材”的视频也会带封面并在刷新后恢复。
+ [优化] 配置弹窗调用格式去除“官方接口 / 第三方接口”分组，并将 Cai、NewToken、Duomi 等展示名去掉 API 字样。

## v0.4.8 - 2026-06-24

+ [优化] 画布生成输入框点击 `@` 按钮会按当前光标位置打开引用素材候选菜单，不再需要手动输入 `@`。
+ [优化] 画布视频节点连接音频后会在参考缩略图中显示音频图标，便于确认音频素材已连接。
+ [优化] 生图工作台提示词框支持 `@` 引用已添加的参考图，并在提交时继续把参考图作为图片文件传给上游。

## v0.4.7 - 2026-06-23

+ [新增] 画布脚本生成器支持“主题分镜 / 参考图文案 / 图文转视频”三种模式，支持上传参考图后自动连线，并在生成结果中保留文案分析摘要。
+ [优化] Cai / NewToken / duomiapi 视频参考图片提交时统一重新上传到本服务临时存储，不再直传原始外部图片 URL。
+ [优化] 参考素材临时上传接口补充 `HEAD`、`Content-Length`、`Accept-Ranges` 等响应头，并在提交上游前自检公网 URL 可读性；当反向代理未返回 `Content-Length` 或拦截 `HEAD` 时，会改用 `GET Range` 探测兜底。
+ [优化] 生图工作台和视频创作台支持已有任务生成中继续提交新任务，历史记录保留每次提交，视频任务可并行后台轮询。
+ [优化] Cai API 的 `videos` / `videos_stable` 按 Seedance 能力处理，支持首尾帧、全能参考和参考视频，并在配置页提供 Cai 视频模型一键填入。
+ [修复] `grok-imagine-video-1.5-preview` 在 Cai 专用接口、NewToken 和 duomiapi 下统一使用 `input_reference` 提交参考图，并在未连接图片时先给出本地提示。
+ [修复] 修复视频创作台残留旧 `setRunning` 状态调用导致并发提交状态不完整的问题。
+ [修复] 全能参考模式补充音频单独使用校验，并统一支持模型的提示文案，避免误提交无效参考组合。

## v0.4.6 - 2026-06-22

+ [新增] NewToken `gpt-image2-1k` / `gpt-image2-2k` / `gpt-image2-4k` 改为按官方异步图片任务协议走 `/v1/videos` 创建和轮询，`*_sync` 模型继续使用同步图片接口。
+ [优化] 画布视频提交与视频创作台校验对齐，统一空提示词拦截、参考素材校验、比例/清晰度/秒数/声音/水印参数归一化。
+ [优化] Cai / NewToken / duomiapi 视频参考图片、视频和音频在提交前会上传为公网 HTTPS URL，减少本地 `blob:` / dataURL 导致的上游超时。
+ [优化] 画布缩小时保持生成输入面板可读，并提高视频输入区最小高度。
+ [修复] 修复画布视频提交时误用未定义变量导致 `node is not defined` 的问题。
+ [修复] 视频最大生成时长统一限制为 15 秒，避免提交上游不支持的更长时长。
